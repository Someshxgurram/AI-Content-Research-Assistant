# AI-Content-Research-Assistant

A lightweight multi-agent AI workflow built using FastAPI and OpenAI APIs.

The system simulates multiple AI agents:
- Research Agent
- LinkedIn Content Agent
- Twitter Thread Agent

It generates growth-focused content ideas and viral post suggestions from a single topic input.
